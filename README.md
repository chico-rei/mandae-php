This is a PHP client wrapper for [Mandaê API v2](https://dev.mandae.com.br/api/index.html).

## Install

Via [Composer](https://getcomposer.org/)

```bash
$ composer require chico-rei/mandae-php
```

Requires PHP 7.1 or newer.

## Testing

```bash
$ composer test
```
